src.camera module¶

src.camera.begin_camera()[source]¶

Begins the camera transformation.

This function pushes the current matrix stack down by one, duplicating the current matrix. Then it applies a translation transformation to the matrix using the current camera offsets.

Returns:

None

Global Variables:

  • offset_x (int): The current x-offset of the camera.

  • offset_y (int): The current y-offset of the camera.

src.camera.end_camera()[source]¶

Ends the camera transformation.

This function pops the current matrix stack up by one, removing the current matrix.

Returns:

None
